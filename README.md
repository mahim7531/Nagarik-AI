# Nagarik-AI
Build a full-stack MERN (MongoDB, Express.js, React.js, Node.js) web application named "Citizen Connect Bangladesh".

Goal:
Create a smart civic engagement platform that helps Bangladesh citizens directly report public problems to government authorities and local administrations.

Core Features:

1. Road Damage Reporting System

* Citizens can upload photos of damaged roads, broken bridges, potholes, drainage issues, and street infrastructure problems.
* Automatically capture GPS location.
* AI image analysis to classify damage severity (Low, Medium, High, Critical).
* Generate a report and send it to the appropriate government department dashboard.
* Track report status (Pending, Verified, In Progress, Resolved).
* Citizens receive notifications when status changes.

2. Missing, Mentally Challenged & Lost Person Reporting

* User can upload image of a lost person, mentally challenged person, elderly person, or anyone unable to identify their home.
* Capture live location.
* Automatically notify nearest police station and local administration.
* Face recognition support for matching with existing missing-person records.
* Emergency reporting button.
* Real-time report tracking.

3. City Problem Reporting Module
